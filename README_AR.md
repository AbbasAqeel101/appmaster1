# تطبيق AppMaster (المعدل من Feather)

تطبيق **AppMaster** هو متجر تطبيقات متكامل لنظام iOS، مبني على كود **Feather v2.7.0** مفتوح المصدر. تم تعديله ليكون متجراً مفتوحاً للجميع بدون قيود الـ UDID، مما يتيح لأي مستخدم تثبيت تطبيقاته الخاصة باستخدام شهادته الشخصية.

## مميزات التطبيق:
- **واجهة عربية بالكامل**: تم تعريب جميع القوائم والإعدادات لتسهيل الاستخدام.
- **دعم الشهادات**: يمكنك استيراد شهاداتك الخاصة وتوقيع التطبيقات بسهولة.
- **إدارة المصادر**: إضافة مستودعات تطبيقات (Repos) خارجية لتحميل التطبيقات مباشرة.
- **بدون قيود UDID**: التطبيق يعمل للجميع ولا يتطلب تسجيل جهازك في سيرفر خاص.
- **دعم ملفات IPA و TIPA**: تثبيت التطبيقات بصيغ مختلفة.
- **تخصيص كامل**: إمكانية تغيير الأيقونات والمظهر (داكن/فاتح).

## كيفية إضافة تطبيقاتك:
يمكنك إنشاء ملف `json` يحتوي على روابط تطبيقاتك ورفعه على أي استضافة (مثل GitHub Pages)، ثم إضافة الرابط في قسم "المصادر" داخل التطبيق.

## ملاحظة:
هذا الإصدار معدل ليكون مثل تطبيق **E-Sign** في سهولة الاستخدام والحرية في التثبيت.
